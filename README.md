# Netflix-Movie-Data-Analysis

## üöÄ Project Overview  
This repository presents a data analysis project on Netflix movie data. The core objective is to explore, visualize, and derive insights from a dataset of movies on the streaming platform, focusing on patterns, correlations and key metrics (e.g., by genre, release year, runtime, ratings). The analysis is conducted using a Jupyter Notebook and accompanying dataset.

## üìä Contents  
- `mymoviedb.csv` ‚Äì the dataset containing Netflix movie information (attributes such as title, genre, release year, runtime, IMDb/Netflix rating, etc).  
- `Netflix Movie Data analysis.ipynb` ‚Äì the Jupyter Notebook housing the data cleaning, exploratory data analysis (EDA), visualization and interpretive commentary.  
- `README.md` ‚Äì this documentation file, explaining the project and how to run/extend it.

## üß† Motivation  
Streaming platforms like Netflix have become ubiquitous, and understanding the content library can reveal interesting insights about trends in genre preference, movie runtime, rating distributions, and more. Through this project you will:  
- Learn how to load, clean and pre-process real-world datasets.  
- Explore relationships between different movie attributes.  
- Build and interpret visualizations (histograms, scatter plots, bar charts).  
- Generate actionable observations: e.g., which genres dominate which years; does runtime correlate with ratings; how ratings vary over time, etc.
